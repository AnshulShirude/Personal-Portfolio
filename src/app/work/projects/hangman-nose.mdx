## About

A unique twist on the classic drawing game concept, where players use their nose 👃 as a drawing tool to recreate words, combining facial recognition technology with creative gameplay.

### Key Features

- **Drawing Mechanics**

  - Nose-based drawing interface 👃
  - Real-time facial tracking 🙂
  - Multiple color options 🎨
  - Eraser functionality

- **Game Features**
  - 5-letter word dictionary 📚
  - Progressive letter reveal system 🔠
  - Gesture-based controls 🖐️
  - Multiplayer support

### Technical Implementation

- **Computer Vision**

  - OpenCV integration
  - Facial feature detection
  - Real-time tracking
  - Gesture recognition

- **Game Logic**
  - Word selection system
  - Score tracking
  - Drawing validation
  - Multi-player synchronization

### Interactive Features

1. **Drawing Tools**

   - Color palette selection 🎨
   - Eraser tool
   - Line thickness control
   - Clear canvas option

2. **Game Controls**

   - Gesture-based commands 🖐️
   - Word skip functionality
   - Letter reveal system
   - Score tracking

3. **User Interface**
   - Real-time drawing preview
   - Word display
   - Timer system
   - Player status indicators

### Tech Stack

- Python
- OpenCV
- Sys Module
- Custom Dictionary Implementation

The game was developed over a three-month period, focusing on:

2. **Game Design**

   - Interface optimization

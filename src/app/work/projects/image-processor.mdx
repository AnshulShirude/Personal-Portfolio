## About

A robust Java image processing application that combines text-based ⌨️, Graphical User Interface (GUI) 🗾, and command-line functionality to provide comprehensive image editing capabilities.

### Key Features

- **File Operations**

  - Multi-format support for .jpg, .png, .ppm, and .bmp files 💾
  - Image upload and save functionality 🌠
  - Cross-format conversion support

- **Image Manipulations**

  - Brightness/contrast adjustment ☀️
  - Vertical and horizontal flipping
  - Greyscale conversion
  - Blur and sharpen filters
  - Sepia filter application
  - Image downsizing

- **GUI Features**
  - Interactive histogram visualization 📊
  - Real-time RGB value frequency display
    - Red component visualization 🟥
    - Green component visualization 🟩
    - Blue component visualization 🟦
  - Value range analysis (0-255)

### Technical Implementation

- **Architecture**

  - Model-View-Controller (MVC) design pattern
  - Interface-driven development following object-oriented programming principles

- **Testing**
  - Comprehensive JUnit4 test suite 📚
  - Edge case coverage
  - Feature validation
  - Integration testing

### Tech Stack

- Java 8
- Java Swing (GUI)
- JUnit4 (Testing)

The application was built following solid software engineering principles:
